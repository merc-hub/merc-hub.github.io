- Personal introduction paragraph

## How to Use

### Opening the Website
1. Open `home-index.html` in a web browser to start at the home page
2. Use the navigation menu to move between pages
3. All pages are fully functional and linked together

### Customization

**Change Name/Title:**
- Search for "Mercy Kimeu" and replace with your name
- Update the hero tagline to reflect your focus

**Update Contact Information:**
- Edit email, phone, and timezone in contact-page.html
- Update social media links to your profiles

**Add Projects:**
- Duplicate a project card in projects.html
- Update title, description, and tags
- Add image link when images are available

**Modify Skills:**
- Edit skill categories in about.html
- Update skill lists to match your expertise

**Change Content:**
- Update personal story in about.html
- Modify project descriptions
- Change timeline entries
